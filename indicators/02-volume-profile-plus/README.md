# Volume Profile Plus

> 🚧 **Planned — not yet released.**

Session-based volume profile for Pine Script v6: POC, VAH/VAL, high/low volume nodes, and developing-value tracking.

Planned scope:

- Session, daily, and weekly profile anchoring
- Point of Control (POC) + Value Area High/Low (70% by default, configurable)
- High-volume / low-volume node detection for support-resistance context
- Naked POC tracking (untested prior-session POCs)

Until this ships, a lightweight rolling-POC proxy is available inside the [Pro Confluence Engine](../../pro/README.md)'s volume component.
