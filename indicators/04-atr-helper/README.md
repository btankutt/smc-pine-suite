# ATR Helper

> 🚧 **Planned — not yet released.**

Dynamic SL/TP and position-sizing helper for Pine Script v6.

Planned scope:

- ATR-based and structure-based stop-loss suggestions with configurable buffers
- R:R-derived take-profit levels with on-chart R multiples
- Position-size calculator (account size x risk % / stop distance)
- Clean line/label visualization for the active setup

Until this ships, the same risk logic (structural SL with ATR floor, R:R targets, size hint) is built into the [Pro Confluence Engine](../../pro/README.md)'s risk module.

Follow the [roadmap](../../README.md#roadmap) for status.
