# Pricing & Access

> The Pro Confluence Engine is **invite-only** on TradingView. The source is not distributed publicly (see [LICENSE-PRO](../LICENSE-PRO)). This page explains how access works.

## How invite-only access works

1. You request access (see below).
2. Your TradingView username is added to the script's allowed list.
3. The indicator appears under **Invite-only scripts** in your TradingView account.
4. You add it to any chart like any other indicator — no source code is shared.

Access is per-TradingView-account and non-transferable.

## FAQ

**Do I get the source code?**
No. Invite-only scripts run on TradingView without exposing source. Everything the engine *scores*, however, is implemented openly in the [free indicators](../indicators) so the methodology is auditable.

**Can I use it on multiple accounts?**
Access is tied to one TradingView account. Contact for multi-seat arrangements.

**Is there a refund policy / trial?**
Discuss terms when you request access.

**Why isn't the free suite enough?**
It can be — the four free indicators are fully functional. The Pro engine's value is automating the *fusion and adaptive weighting* of all four into one score, plus presets and updates.
